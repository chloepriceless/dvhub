The `gracefulShutdown` function must call stop/close on ALL modules:
```javascript
async function gracefulShutdown(signal) {
  console.log(`\n${signal} received, shutting down...`);
  poller.stop();
  scheduler.stop();
  liveTelemetryBuffer?.flush({ force: true });
  epex.stop();
  if (runtimeWorker) runtimeWorker.kill();
  await Promise.all([transport.destroy(), scanTransport.destroy()]);
  if (telemetryStore) telemetryStore.close();
  modbus.close();
  if (IS_WEB_PROCESS) web.close();
  setTimeout(() => process.exit(0), 500).unref();
}
```
Verify this list is complete. Note: routes module has no lifecycle (no start/stop) -- this is correct, no change needed.
