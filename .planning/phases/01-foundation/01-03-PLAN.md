**dvhub/modules/gateway/modbus-proxy.js:**

Import `net` from `node:net`.

Export `function createModbusProxy({ config, eventBus, log })`:

Constants:
- `const MAX_BUFFER_SIZE = 1024` (Modbus TCP frame cap -- standard Modbus TCP max PDU is 260 bytes, 1024 is generous)
- `const bindHost = config.modbusListenHost || '127.0.0.1'` (SECURITY FIX: default was 0.0.0.0)
- `const bindPort = config.modbusListenPort || 1502`
- `const allowList = new Set(config.modbusAllowList || [])`

Creates `net.createServer((socket) => { ... })`:

On connection:
1. Extract `remoteIp = socket.remoteAddress?.replace('::ffff:', '')` (handle IPv4-mapped IPv6)
2. If `allowList.size > 0 && !allowList.has(remoteIp)`: log warning `{ remoteIp }` 'Modbus connection rejected: IP not in allowlist', call `socket.destroy()`, return.
3. Log info: `{ remoteIp }` 'Modbus client connected'
4. Initialize `let buffer = Buffer.alloc(0)`
5. On `socket.on('data', (chunk))`:
   a. `buffer = Buffer.concat([buffer, chunk])`
   b. If `buffer.length > MAX_BUFFER_SIZE`: log warn `{ remoteIp, bufferSize: buffer.length }` 'Modbus buffer size exceeded', `socket.destroy()`, return.
   c. Process complete Modbus TCP frames from buffer. A Modbus TCP frame has a 6-byte MBAP header where bytes 4-5 (big-endian) give the remaining length. If buffer has enough bytes for a complete frame, extract it, process it via a `processFrame(frame, socket)` callback, and remove processed bytes from buffer.
6. On `socket.on('close')`: log info 'Modbus client disconnected'
7. On `socket.on('error', (err))`: log warn `{ err }` 'Modbus socket error'

The `processFrame` function is passed in via options or set via a `setFrameHandler(fn)` method, since the actual Modbus frame processing logic (FC3/FC4/FC6/FC16 from server.js lines 1025-1084) will be wired in Plan 04. For now, the proxy just manages connections and security.

Return object:
- `start()`: returns `new Promise((resolve) => server.listen(bindPort, bindHost, resolve))`
- `stop()`: returns `new Promise((resolve) => server.close(resolve))`
- `setFrameHandler(fn)`: sets the frame processing callback
- `getConfig()`: returns `{ host: bindHost, port: bindPort, allowListSize: allowList.size }`

Write tests FIRST. Use `net.connect()` to simulate Modbus clients.
