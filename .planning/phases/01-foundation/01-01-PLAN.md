**dvhub/core/module-registry.js:**
Export `createModuleRegistry()` returning object with:
- `register(mod)` - stores module in a Map keyed by `mod.name`. Throws if `mod.name` is missing.
- `async initAll(ctx)` - validates dependencies (each `mod.requires[]` must exist in the Map), then calls `await mod.init(ctx)` in Map insertion order (Gateway registered first).
- `async destroyAll()` - calls `await mod.destroy()` in reverse order. Each destroy is wrapped in try/catch to ensure all modules get destroyed even if one throws.
- `get(name)` - returns module by name or undefined
- `getAll()` - returns array of all registered modules
- `has(name)` - returns boolean

**dvhub/core/event-bus.js:**
Import `{ BehaviorSubject, Subject, filter }` from `rxjs`.
Export `createEventBus()` returning object with:
- `createStream(name, initialValue)` - creates BehaviorSubject, stores in Map, returns it
- `getStream(name)` - returns BehaviorSubject or undefined
- `getValue(name)` - calls `streams.get(name).getValue()`. Throws `Error('Stream "${name}" not registered')` if stream doesn't exist. This is the synchronous read path for DV real-time.
- `emit(event)` - calls `events.next(event)` on a generic Subject
- `on$(type)` - returns `events.pipe(filter(e => e.type === type))`
- `destroy()` - calls `.complete()` on all streams and the events Subject

**Tests:** Use `node:test` and `node:assert`. Write tests FIRST (RED), then implement (GREEN).
