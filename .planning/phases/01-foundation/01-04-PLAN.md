**dvhub/server.js (new bootstrap, target ~200 lines):**

The new server.js follows the pattern from 01-RESEARCH.md "Bootstrap server.js" code example:

```javascript
// dvhub/server.js — v2 Bootstrap
import Fastify from 'fastify';
import { loadConfig } from './core/config.js';
import { createEventBus } from './core/event-bus.js';
import { createModuleRegistry } from './core/module-registry.js';
import authPlugin from './core/auth.js';
import { createGatewayModule } from './modules/gateway/index.js';
```

Steps:
1. Determine config path from `process.env.DV_APP_CONFIG` or default to `new URL('config.json', import.meta.url).pathname`
2. Call `loadConfig(configPath)` to get `{ rawConfig, config }` (validates module activation)
3. Create Fastify instance with Pino logging:
   - `level: config.logLevel || 'info'`
   - In development (`NODE_ENV=development`): add pino-pretty transport
4. Create `eventBus = createEventBus()`
5. Create `registry = createModuleRegistry()`
6. Register cross-cutting plugins:
   - `await fastify.register(authPlugin, { apiToken: config.apiToken, roles: config.roles })`
   - `await fastify.register(import('@fastify/websocket'))`
7. Create and register Gateway module (always active):
   - `const gateway = createGatewayModule(config)`
   - `registry.register(gateway)`
8. Conditionally register DV and Optimizer modules:
   - `if (config.modules?.dv?.enabled)` -> import and register DV module
   - `if (config.modules?.optimizer?.enabled)` -> import and register Optimizer module
9. Initialize all modules: `await registry.initAll({ fastify, eventBus, config, rawConfig })`
10. Register module Fastify plugins:
    ```javascript
    for (const mod of registry.getAll()) {
      if (mod.plugin) await fastify.register(mod.plugin);
    }
    ```
11. Start Fastify: `await fastify.listen({ port: config.httpPort || 3000, host: '0.0.0.0' })`
12. Graceful shutdown handlers for SIGINT and SIGTERM:
    - Log signal
    - `await registry.destroyAll()`
    - `eventBus.destroy()`
    - `await fastify.close()`
    - `process.exit(0)`

The file MUST be under 250 lines.
